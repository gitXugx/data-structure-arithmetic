# 栈
> 栈属于具有权限的线性表, 它只能从栈顶进行删除和插入操作, 并且不支持随机访问。

## 什么是栈
栈, `先进后出` 或 `后进先出` 只能操作栈顶的数据结构

栈的实现有很多方式:
1. 数组的方式实现固定大小的栈
2. 可变数组实现栈
3. 链表实现栈
   

## 栈的操作

**数组实现的栈**
`pop` 是删除栈顶元素, 最差是移动n个, 所以他的时间复杂度 `O(n)`

|:------:|:---:|:--:|
